# Estrutura do Projeto ThinkCoffee

## Padrão de Arquitetura
- **Monorepo com PNPM Workspaces**

## Estrutura de Pastas
- `packages/core`: Lógica de negócio e tipos compartilhados.
- `packages/mcp-server`: Servidor backend (Node.js/Express).
- `packages/vscode`: Extensão para Visual Studio Code.
- `packages/cli`: Interface de linha de comando.

## Tecnologias Utilizadas
- **Linguagem:** TypeScript
- **Backend:** Node.js, Express.js, SQLite
- **Testes:** Vitest

## Como Contribuir
1. Clone o repositório.
2. Instale as dependências com `pnpm install`.
3. Execute os testes com `pnpm test`.
