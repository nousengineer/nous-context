## 🚀 COMO USAR

**REST API:**
```bash
curl -H "Authorization: Bearer TOKEN" \
  http://localhost:3000/api/services/metrics/

curl -X POST -H "Authorization: Bearer TOKEN" \
  http://localhost:3000/api/services/triggers/event \
  -d '{"workflowId":"wf-1","eventType":"pipeline:completed"}'
```

**CLI:**
```bash
# Monitorar métricas
think metrics get day

# Criar trigger de evento
think trigger register-event --workflow wf-1 --event pipeline:completed

# Executar workflow
think workflow execute --id wf-1 --params '{"param":"value"}'

# Ver diagnósticos
think diagnostic list --pipeline pipeline-1
```
